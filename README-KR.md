# Obsidian Excalidraw – S펜 버튼 임시 지우개 (Freedraw 전용)

[데모 영상](https://youtu.be/BtNIPnEEaXg)

이 스크립트는 Obsidian의 Excalidraw에서 **S펜 버튼을 눌러 임시로 지우개로 전환**할 수 있도록 도와주는 Automate 스크립트입니다.

## 개요

Excalidraw의 펜 모드에서 더블탭으로 지우개 전환이 가능하지만, 반복해서 사용하기에는 불편했습니다. 그래서 “**그리기 → S펜 버튼 → 지우기 → 다시 그리기**”의 흐름을 자연스럽게 만들고자 이 스크립트를 제작했습니다.

현재는 **삼성 태블릿(S펜)** 환경에서만 테스트되었기 때문에 다른 환경에서는 동작이 다를 수 있습니다.

또한 S펜의 구조상, **버튼을 누르는 것만으로 지우개 전환은 기술적으로 불가능**합니다. 그래서 **버튼 + 화면을 한 번 터치/드래그**하는 동작을 트리거로 사용합니다. 마우스 우클릭 지원은 제한적이며, 이는 S펜 중심 환경을 기준으로 설계되었기 때문입니다.

---

## ✨ 기능 요약

- **Freedraw(자유 그리기) 도구 사용 중에만 작동**
- S펜 버튼을 누른 상태에서:
  - 화면을 한 번 터치하거나 살짝 긁으면 → **즉시 지우개로 전환**
- 지우기 후 펜을 떼면:
  - 자동으로 **원래 Freedraw 도구로 복귀**

---

## 🖊️ 사용 방법 (S펜 기준)

1. Excalidraw Automate에서  
   **`Toggle Eraser on Right Click in Freedraw`** 스크립트를 실행
2. 드로잉 툴(freedraw) 선택
3. 평소처럼 그림 그리기
4. 지우고 싶을 때:
   - S펜 측면 버튼을 누른 상태에서
   - 화면을 **한 번 터치**하거나 **살짝 드래그**  
     → 지우개 모드 활성화
5. 한 번 지우기 동작이 끝나면:
   - 자동으로 이전 도구(freedraw)로 복귀

---

## 🛠 설치 방법

1. **Excalidraw 플러그인 설치 여부 확인**
2. Obsidian 설정으로 이동  
   **설정 → 커뮤니티 플러그인 → Excalidraw**
3. Basic 탭에서  
   **Excalidraw Automate script folder** 경로 확인
4. 이 저장소의 파일들(README는 제외)을 해당 폴더에 그대로 복사  
   (**대소문자 구분에 주의!**)

---

## ⚠️ 참고 사항

- S펜 사용자에게 최적화된 기능입니다.
- 마우스 환경에서는 일부 동작이 정상적이지 않을 수 있습니다.
- Freedraw가 아닐 때는 자동 전환이 일어나지 않습니다.
- 스크립트를 다시 실행하면 이전 리스너가 제거되어 중복 등록을 방지합니다.
